## Reels Description

Upload videos by nestjs framework

## Installation

```bash
npm install
```

## Env
copy .env.example and rename to .env <br>
Set variables

## Running the app

```bash

# buils project
npm run build

# create database
docker-compose up -d

# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```
